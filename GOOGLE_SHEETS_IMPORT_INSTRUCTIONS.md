# How to Import final_evolutions_data_changes.csv into Google Sheets

## Method 1: Direct Import (Recommended)
1. Open Google Sheets
2. Go to **File** â†’ **Import**
3. Click **Upload** tab
4. Select `final_evolutions_data_changes.csv`
5. Choose import settings:
   - **Import location**: Replace spreadsheet / Insert new sheet
   - **Separator type**: Comma
   - **Convert text to numbers, dates, and formulas**: Yes
6. Click **Import data**

## Method 2: Copy and Paste
1. Open `final_evolutions_data_changes.csv` in a text editor (Notepad, etc.)
2. Select all (Ctrl+A) and copy (Ctrl+C)
3. In Google Sheets, click on cell A1
4. Paste (Ctrl+V)
5. Google Sheets should automatically detect the columns

## Method 3: Drag and Drop
1. Open Google Sheets
2. Simply drag the `final_evolutions_data_changes.csv` file into the browser window
3. Google Sheets will automatically import it
